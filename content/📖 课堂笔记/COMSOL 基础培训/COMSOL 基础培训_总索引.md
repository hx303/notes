## 训练营 1：基本建模流程与多物理场耦合

> 🔗 详见：[01 基本建模流程](01%20基本建模流程)

**核心主题：**
- COMSOL 产品体系与多物理场仿真平台
- 建模前思考方法（简化、物理过程、耦合、边界条件、分析类型）
- 演示案例：微阻梁电致热变形（电场 → 焦耳热 → 热膨胀）
- 模型向导与建模工作流（定义 → 几何 → 材料 → 物理场 → 网格 → 研究 → 结果）
- 物理场设置（电流/固体传热/固体力学 + 边界条件替代关系）
- 多物理场耦合（内置耦合节点 + 手动耦合原理 + 方程视图）
- 参数定义的优势（集中管理/参数化扫描/App 封装）
- 参数化扫描（所有组合 vs 指定组合，Range 函数）
- 辅助扫描（逐步加载，不同参数取值间有联系）
- 研究类型选择（稳态/瞬态/频域稳态/频域瞬态）
- 优化功能（参数优化、形状优化、拓扑优化）
- 仿真 App 开发器（无代码创建 → Compiler 部署）

## 训练营 2：几何建模

> 🔗 详见：[02 几何建模](02 几何建模)

**核心主题：**
- 几何内核（COMSOL 内核 vs CAD Parasolid 内核）
- 几何对象体系（域/边界/边/点，三维/二维/一维）
- 几何序列与节点操作
- 基本体素（长方体、圆柱、球体、螺旋等）
- 二维→三维：拉伸、回转、扫掠、放样
- 三维→二维：横截面提取
- 参数化曲线与曲面
- 布尔操作（并集/差集/交集）
- 变换操作（移动/缩放/阵列/镜像）
- 几何导入、修复与清理
- Live Link 接口（CAD 实时同步）
- 草图建模工具

## 训练营 3：网格划分

> 🔗 详见：[03 网格划分](03 网格划分)

**核心主题：**
- 网格基础知识（离散化、质量、精度权衡）
- 单元类型（四面体/六面体/棱柱/金字塔/三角形/四边形）
- 物理场控制网格（9 级预定义尺寸）
- 用户控制网格（网格序列）
- 网格尺寸控制参数（最大/最小尺寸、增长率、曲率因子、狭窄区域分辨率）
- 结构化网格（映射 + 扫掠）
- 非结构化网格（自由四面体/三角形）
- 边界层网格
- 复制网格与相同网格约束
- 网格质量评价与可视化
- 自适应网格细化
- 动网格与变形几何

## 训练营 4：结果处理与可视化

> 🔗 详见：[04 结果处理与可视化](04%20结果处理与可视化)

**核心主题：**
- 结果节点体系（数据集 → 派生值 → 表格 → 绘图组 → 导出 → 报告）
- 数据集操作（解数据集、截面/切面/选择、等值面/等值线、二维/三维转换、合并、嵌套）
- 派生值计算（体积分/面积分/线积分/平均值/最大最小值/全局计算）
- 表格功能（结果储存、更新策略、表图快速绘制）
- 绘图组体系（三维/二维/一维/极坐标/史密斯绘图组）
- 绘图类型（体图/面图/线图、等值面、切面/多切面、流线图、箭头图、标注图）
- 绘图增强（变形显示、材料外观、透明度、颜色表达式、过滤器）
- 结果导出（图像/数据/动画：播放器 → 文件）
- 报告创建（Word/PPT 自动生成，三级模板）

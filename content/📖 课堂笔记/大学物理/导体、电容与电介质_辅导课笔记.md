## 一、导体与静电平衡（复习深化）

### 1.1 静电感应过程

将一个导体放入外电场 $\vec{E}_0$ 中：

1. 导体内的**自由电子**在电场力作用下移动
2. 负电荷向与 $\vec{E}_0$ 相反的方向聚集
3. 正负电荷分离 → 导体内部产生**感应电场** $\vec{E}'$
4. 感应电场与外电场方向相反
5. 当 $\vec{E}_{\text{内}} = \vec{E}_0 + \vec{E}' = 0$ 时 → **静电平衡**

> 💡 **关键理解**：静电平衡时导体内部场强为零，这是自由电子重新分布后的必然结果。

### 1.2 静电平衡的特征

| 特征 | 说明 |
|------|------|
| 内部场强 | $\vec{E} = 0$ |
| 导体是等势体 | 内部和表面各处电势相等 |
| 表面是等势面 | $\vec{E}$ 始终垂直于导体表面 |
| 电荷分布 | **仅分布在导体表面**（内部无净电荷） |
| 表面场强 | $E = \dfrac{\sigma}{\varepsilon_0}$（紧贴表面的外侧） |

### 1.3 导体表面场强的两种表达式辨析

这是最容易混淆的地方：

| 公式 | 适用场景 |
|------|----------|
| $E = \dfrac{\sigma}{\varepsilon_0}$ | **导体表面附近**（一面有场，内部 $E=0$） |
| $E = \dfrac{\sigma}{2\varepsilon_0}$ | **无限大均匀带电平面**（两面都有场） |

**统一理解（叠加原理）**：

将导体表面一小块 $\Delta S$ 视为无限大带电平面：
- $\Delta S$ 自身产生的场：$\sigma/(2\varepsilon_0)$，两面各半
- 除 $\Delta S$ 外的其他所有电荷产生的场：$E_{\text{other}}$
- 导体内：$E_{\text{other}} - \frac{\sigma}{2\varepsilon_0} = 0$ → $E_{\text{other}} = \frac{\sigma}{2\varepsilon_0}$
- 导体外：$E_{\text{总}} = E_{\text{other}} + \frac{\sigma}{2\varepsilon_0} = \frac{\sigma}{\varepsilon_0}$ ✓

> 🔑 两者不矛盾——叠加原理统一了它们！

### 1.4 导体表面电荷分布与曲率

**定性规律**：曲率越大（越尖锐），电荷密度越大。

- 尖端的曲率半径 $R$ 极小 → 电荷密度 $\sigma$ 极大
- 尖端附近的电场极强 → **尖端放电**（电晕放电）
- 避雷针的原理正是利用尖端放电

> 📌 定量关系：对于孤立椭球导体，$\sigma \propto 1/\text{曲率半径}$

### 1.5 导体空腔

**腔内有电荷**：
- 内表面感应出等量异号电荷 $(-Q)$
- 外表面感应出等量同号电荷 $(+Q)$
- 用高斯定理可严格证明

**腔内无电荷**：
- 内表面无净电荷
- 腔内 $E = 0$（**静电屏蔽**）

### 1.6 接地问题

- 接地 → 导体电势与大地电势相等（取 $V = 0$）
- 接地的实质：导体与大地交换电荷，直至等势
- 常见问题：接地导体附近有带电体时，导体表面会产生感应电荷

### 1.7 静电平衡的建立时间

对于良导体（如铜），弛豫时间：

$$\tau = \frac{\varepsilon_0}{\sigma_c} \approx 1.5 \times 10^{-19}\ \text{s}$$

> 极快！宏观上可视为「瞬间」完成。

## 二、电容与电容器

### 2.1 电容的定义

导体的电容：

$$C = \frac{Q}{V}$$

- $Q$：导体所带电荷量（绝对值）
- $V$：导体的电势（取无穷远为电势零点）
- 单位：法拉（F），$1\ \text{F} = 1\ \text{C/V}$

> ⚠️ 法拉是非常大的单位。实际器件中常用：
> - 微法：$1\ \mu\text{F} = 10^{-6}\ \text{F}$
> - 皮法：$1\ \text{pF} = 10^{-12}\ \text{F}$

**电容的物理意义**：反映导体**容纳电荷的能力**。电容越大，升高单位电势所需的电荷越多。

### 2.2 孤立导体的电容

**孤立导体球**（半径 $R$）：

$$V = \frac{Q}{4\pi\varepsilon_0 R} \quad\Rightarrow\quad C = \frac{Q}{V} = 4\pi\varepsilon_0 R$$

> 例：地球半径 $R \approx 6.4 \times 10^6\ \text{m}$，$C \approx 7 \times 10^{-4}\ \text{F} = 700\ \mu\text{F}$

### 2.3 电容器的定义

两个导体组成的系统，彼此绝缘：
- 一个带 $+Q$，另一个带 $-Q$
- 电容：$C = \dfrac{Q}{U}$，其中 $U$ 为两导体间的电势差（电压）

## 三、常见电容器的电容公式

### 3.1 平行板电容器

![](示意图：两块面积A、间距d的平行金属板)

$$C = \frac{\varepsilon_0 A}{d}$$

**推导**：
1. 两极板间为匀强电场：$E = \dfrac{\sigma}{\varepsilon_0} = \dfrac{Q}{\varepsilon_0 A}$
2. 电势差：$U = Ed = \dfrac{Qd}{\varepsilon_0 A}$
3. $C = \dfrac{Q}{U} = \dfrac{\varepsilon_0 A}{d}$

> 📌 **讨论**：$A$ 越大、$d$ 越小 → 电容越大。

### 3.2 圆柱形电容器

（同轴圆柱，内半径 $R_1$，外半径 $R_2$，长度 $L$，且 $L \gg R_2$）

$$C = \frac{2\pi\varepsilon_0 L}{\ln(R_2/R_1)}$$

**推导思路**：
1. 用高斯定理求两柱面间电场：$E(r) = \dfrac{\lambda}{2\pi\varepsilon_0 r} = \dfrac{Q}{2\pi\varepsilon_0 L r}$
2. 电势差：$U = \int_{R_1}^{R_2} E\,dr = \dfrac{Q}{2\pi\varepsilon_0 L}\ln\dfrac{R_2}{R_1}$
3. $C = \dfrac{Q}{U} = \dfrac{2\pi\varepsilon_0 L}{\ln(R_2/R_1)}$

### 3.3 球形电容器

（同心球壳，内半径 $R_1$，外半径 $R_2$）

$$C = \frac{4\pi\varepsilon_0}{\dfrac{1}{R_1} - \dfrac{1}{R_2}} = \frac{4\pi\varepsilon_0 R_1 R_2}{R_2 - R_1}$$

**极限情况**：
- 当 $R_2 \to \infty$：$C = 4\pi\varepsilon_0 R_1$ → 回归孤立导体球的电容 ✓

## 四、电容器的串联与并联

### 4.1 串联

![串联电容器]

**特征**：各电容器极板上电荷量相等（$Q_1 = Q_2 = \cdots = Q$）

$$\frac{1}{C_{\text{总}}} = \frac{1}{C_1} + \frac{1}{C_2} + \cdots + \frac{1}{C_n}$$

**电压分配**：
$$U_i = \frac{Q}{C_i}, \quad U = U_1 + U_2 + \cdots = Q\left(\frac{1}{C_1} + \frac{1}{C_2} + \cdots\right)$$

> 📌 串联后**总电容减小**（小于任一个单独电容）。类似于电阻的并联。

### 4.2 并联

![并联电容器]

**特征**：各电容器两端电压相等（$U_1 = U_2 = \cdots = U$）

$$C_{\text{总}} = C_1 + C_2 + \cdots + C_n$$

**电荷分配**：
$$Q_i = C_i U, \quad Q = Q_1 + Q_2 + \cdots = (C_1 + C_2 + \cdots)U$$

> 📌 并联后**总电容增大**（等于各电容之和）。类似于电阻的串联。

### 4.3 串并联对比

| | 串联 | 并联 |
|------|------|------|
| 总电容公式 | $\dfrac{1}{C} = \sum \dfrac{1}{C_i}$ | $C = \sum C_i$ |
| 不变物理量 | **电荷 $Q$** 相同 | **电压 $U$** 相同 |
| 总电容大小 | 小于最小者 | 大于最大者 |
| 类比电阻 | 并联 | 串联 |

> ⚠️ 电容的串并联公式与电阻**恰好相反**！不要搞混。

## 五、电容器的能量

### 5.1 电容器储能

$$W = \frac{1}{2}CU^2 = \frac{1}{2}\frac{Q^2}{C} = \frac{1}{2}QU$$

**推导**（充电过程）：
1. 充电过程中，极板间电压 $u = q/C$（$q$ 为当前电荷量）
2. 搬运 $dq$ 电荷做的功：$dW = u\,dq = \dfrac{q}{C}\,dq$
3. 从 $0$ 充电到 $Q$：$W = \int_0^Q \dfrac{q}{C}\,dq = \dfrac{Q^2}{2C} = \dfrac{1}{2}CU^2$

### 5.2 电场能量密度

平行板电容器：$C = \varepsilon_0 A/d$，$U = Ed$

$$W = \frac{1}{2}CU^2 = \frac{1}{2}\frac{\varepsilon_0 A}{d}(Ed)^2 = \frac{1}{2}\varepsilon_0 E^2 \cdot Ad = \frac{1}{2}\varepsilon_0 E^2 \cdot V$$

→ **电场能量密度**：

$$w_e = \frac{W}{V} = \frac{1}{2}\varepsilon_0 E^2$$

> 能量储存在**电场**中，而不是电荷本身。这是场的观点。

## 六、电介质简介

> 📝 本节为辅导课中提及的过渡内容，进入第六章"静电场中的导体和电介质"。

### 6.1 导体 vs 电介质

| | 导体 | 电介质（绝缘体） |
|------|------|------|
| 自由电荷 | 大量自由电子 | 几乎没有 |
| 外电场中的行为 | 静电感应（电荷重新分布，内部 $E=0$） | **极化**（束缚电荷取向，内部 $E \neq 0$） |
| 电场能否穿透 | 不能（静电平衡） | 能（但被削弱） |

### 6.2 电介质对电容的影响

在电容器两极板间插入电介质：

$$C = \varepsilon_r C_0$$

其中 $\varepsilon_r$ 为**相对介电常数**（$\varepsilon_r > 1$），$C_0$ 为真空电容。

- 电介质**增大**电容：$C > C_0$
- 本质原因：电介质极化产生反向电场，削弱极板间电场 → 同样电荷下电势差减小 → $C = Q/U$ 增大

### 6.3 极化机制（定性）

**无极分子**：正负电荷中心重合 → 在外电场中位移极化（感应电偶极矩）

**有极分子**：本身有电偶极矩（如 H₂O）→ 在外电场中取向极化

## 七、典型问题与解题技巧

### 7.1 导体相关判断

> **常见判断题**：
> - ❌ "导体内部 $E=0$ 意味着内部没有电荷" → 错，是**无净电荷**，但仍可能有等量正负电荷
> - ❌ "导体表面各处 $E$ 相等" → 错，$E = \sigma/\varepsilon_0$，$\sigma$ 未必均匀
> - ✅ "静电平衡时导体是等势体" → 对

### 7.2 电容计算通用步骤

1. **假设**两极板带电荷 $\pm Q$
2. 利用高斯定理（或叠加原理）求极板间的**电场分布** $E(r)$
3. 积分求**电势差**：$U = \int \vec{E} \cdot d\vec{l}$
4. 代入 $C = Q/U$，$Q$ 约掉 → 得到仅与几何尺寸和介质有关的 $C$

### 7.3 串并联分析技巧

- 先判断连接方式：并联 → 电压相等；串联 → 电荷相等
- 串联分压公式（类似电阻并联分流）：
  $$U_1 = \frac{C_2}{C_1 + C_2}U_{\text{总}}$$
- 并联分流公式：$Q_i = C_i U$

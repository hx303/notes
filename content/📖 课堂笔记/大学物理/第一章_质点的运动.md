## §1-1 参照系·质点·运动学方程

### 参照系与坐标系

| 概念 | 说明 |
|------|------|
| **参照系** | 选择另一个物体作为描述运动的参考 |
| **坐标系** | 固定在参照系上的定量描述框架（直角坐标、极坐标等）|

> 同一运动，不同参照系描述不同。

### 质点

将物体简化成一个具有质量的**几何点**，忽略大小和形状。

**可简化为质点的情况：**
- 物体尺寸 $\ll$ 研究范围
- 物体只做平动

### 位置矢量（位矢）

$$\vec{r} = x\vec{i} + y\vec{j} + z\vec{k}$$

在直角坐标系中，$P(x, y, z)$。

### 运动方程

描述质点位置随时间变化的函数：

$$\vec{r} = \vec{r}(t), \quad x = x(t), \; y = y(t), \; z = z(t)$$

**轨道方程**：消去参数 $t$ 后得到的轨迹方程。

## §1-2 位移·速度·加速度

### 位移

$$\Delta\vec{r} = \vec{r}_B - \vec{r}_A$$

位移是**矢量**，与路径无关；路程 $\Delta s$ 是标量。

### 速度

| 类型 | 定义 | 公式 |
|------|------|------|
| 平均速度 | $\vec{v} = \dfrac{\Delta\vec{r}}{\Delta t}$ | 矢量 |
| 瞬时速度 | $\vec{v} = \dfrac{d\vec{r}}{dt}$ | 矢量，方向沿切线 |
| 平均速率 | $\bar{v} = \dfrac{\Delta s}{\Delta t}$ | 标量 |
| 瞬时速率 | $v = \dfrac{ds}{dt}$ | 标量 $= |\vec{v}|$ |

**直角坐标系：**
$$\vec{v} = \frac{dx}{dt}\vec{i} + \frac{dy}{dt}\vec{j} + \frac{dz}{dt}\vec{k}$$

### 加速度

$$\vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$$

**直角坐标系：**
$$\vec{a} = \frac{dv_x}{dt}\vec{i} + \frac{dv_y}{dt}\vec{j} + \frac{dv_z}{dt}\vec{k}$$

---

## §1-3 圆周运动·角量描述

### 切向加速度与法向加速度

将加速度分解为两个分量：

$$\vec{a} = \vec{a}_t + \vec{a}_n$$

| 分量 | 大小 | 方向 | 物理意义 |
|------|------|------|---------|
| **切向加速度** $\vec{a}_t$ | $a_t = \dfrac{dv}{dt}$ | 沿切线 | 反映速度**大小**变化 |
| **法向加速度** $\vec{a}_n$ | $a_n = \dfrac{v^2}{\rho}$ | 指向曲率中心 | 反映速度**方向**变化 |

总加速度大小：$a = \sqrt{a_t^2 + a_n^2}$

### 角量描述

| 角量 | 定义 | 关系 |
|------|------|------|
| 角位移 | $\Delta\theta$ | — |
| 角速度 | $\omega = \dfrac{d\theta}{dt}$ | $v = \omega r$ |
| 角加速度 | $\alpha = \dfrac{d\omega}{dt}$ | $a_t = \alpha r$ |

### 圆周运动公式（匀变速）

$$\omega = \omega_0 + \alpha t$$
$$\theta = \theta_0 + \omega_0 t + \frac{1}{2}\alpha t^2$$
$$\omega^2 = \omega_0^2 + 2\alpha(\theta - \theta_0)$$

## §1-4 相对运动

### 伽利略坐标变换

设 $S'$ 系相对 $S$ 系以速度 $\vec{u}$ 运动：

$$\vec{r} = \vec{r}' + \vec{u}t, \quad t = t'$$

### 速度合成（伽利略速度变换）

$$\vec{v} = \vec{v}' + \vec{u}$$

其中 $\vec{v}$ 为绝对速度，$\vec{v}'$ 为相对速度，$\vec{u}$ 为牵连速度。

### 加速度合成

在匀速直线相对运动时：$\vec{a} = \vec{a}'$

## 本章核心公式汇总

| 物理量 | 矢量式 | 说明 |
|--------|--------|------|
| 位置 | $\vec{r}(t)$ | 时间函数 |
| 速度 | $\vec{v} = d\vec{r}/dt$ | 瞬时速度 |
| 加速度 | $\vec{a} = d\vec{v}/dt = d^2\vec{r}/dt^2$ | 瞬时加速度 |
| 切向加速度 | $a_t = dv/dt$ | 速度大小变化率 |
| 法向加速度 | $a_n = v^2/\rho$ | 速度方向变化率 |
| 角速度 | $\omega = d\theta/dt$ | $\vec{v} = \omega \times \vec{r}$ |
| 相对速度 | $\vec{v}_{AB} = \vec{v}_A - \vec{v}_B$ | 两个参照系之间 |

# 元明两代藏地流迁史——藏族人向内地的大规模流动

> 📝 来源：2026-05-07 课堂录音（汉王电纸本语音转文字，已勘误）
> ⚠️ 本转录存在大量语音识别错误，关键人名/地名已逐项校勘

---

## 一、青藏高原早期人类活动追溯

- 青藏高原与内地之间最早的人类活动可追溯至距今约 **16 万年前**
- 2020 年兰州大学张东菊教授在甘肃夏河县发现**丹尼索瓦人（Denisovans）**牙齿化石
- 此发现将青藏高原史前人类活动时间从原先的 4 万年推进到 16 万年
- 丹尼索瓦人主要分布在高纬度、高海拔地区（西伯利亚、青藏高原等北半球大部）
- 其基因中存在**低氧适应的突变**，该基因在今日藏族和夏尔巴人身上仍有留存
- 新石器时代：昌都**卡若遗址**发现与黄河上游相似的文化因素，表明已有频繁交往

## 二、元明两代藏人向内地流动的历史背景

### 2.1 统一王朝国家的建立

- **13 世纪初**：成吉思汗统一蒙古各部，建立大蒙古汗国，先后灭西辽、西夏、金
- **1247 年**：萨迦班智达与阔端在**凉州会谈**，西藏地方被纳入蒙古统治 → 青藏高原与蒙古最早正式联系
- **1260 年**：忽必烈继位，建立元朝，结束唐末以来分裂割据局面
- **1368 年**：朱元璋称帝建明朝，逐步统一全国
- **13—17 世纪**：中国实现长达 400 年的大一统，西藏正式纳入中央王朝统治，为人口流动奠定了政治基础

### 2.2 元明两代对藏传佛教的尊崇

**元代：帝师制度**

- 忽必烈开创**帝师制度**，尊崇藏传佛教（尤以萨迦派为主）
- 帝师地位高于诸王百官，统领全国佛教事务
- 中央设**宣政院**管理全国佛教和西藏地方事务
- 地方各路设**释教总统所**，派驻僧人管理佛教事务

**明代：多封众建**

- 洪武年间朱元璋发诏书招谕朵甘思（川西藏东）、乌斯藏（卫藏，今拉萨日喀则一带）
- 明朝对藏传佛教僧人持尊重态度，认为其有"化凶顽、启人心、抵千军"之作用
- 授予不同封号：法王、教王、灌顶国师、大国师等
- 整个明代共封**三大法王**：
  - **大宝法王**（噶举派 → 原属萨迦派的八思巴，明永乐改封噶举派德银协巴）
  - **大乘法王**（萨迦派·昆泽思巴）
  - **大慈法王**（格鲁派·释迦益西）
- **五大教王**：赞善王、护教王、辅教王、阐教王、阐化王
- 特殊：明武宗自封"**大庆法王**"
- 朝贡制度：对时间、人数、礼仪均有规定，赏赐丰厚 → 刺激藏区积极性

### 2.3 交通条件的改善

- 唐代吐蕃时期：主要通过青海**唐蕃古道**，南北受地理限制
- **元代**：为政令物资通达边疆，大力修建从西藏到内地的**驿站系统**
  - 从大都（北京）→ 拉萨 → 萨迦，交通便利
  - 驿站主要利用者为藏传佛教僧人（记载"十六七人"经常使用）
- **明代**：
  - 明初修复元朝通往藏区的驿道
  - 成化年间，经康区南部连接内地的**朵甘思—乌斯藏道**成为重要通道
  - 朝廷规定使者须从**四川**出发（不走西北洮岷），原因：
    1. 蒙古北退后北方边境频遭骚扰
    2. 茶叶赏赐在四川领取，更为便利
  - 水路：四川→长江→京杭大运河→北京，朝廷承担马匹车船费用

### 2.4 藏传佛教各教派寻求外部支持（内在驱动力）

> **核心逻辑**：吐蕃王朝崩溃后，青藏高原走向政教合一体制，但内部缺乏强大的世俗势力支撑宗教发展。各教派与地方势力相互博弈，均无法压倒对方 → **必须寻求外部强大政治力量的支持**。

| 时期 | 主要教派 | 与中央的联系 |
|------|---------|------------|
| 元代前期 | 萨迦派 | 与元朝宫廷建立紧密的宗教—政治联系，居主导地位 |
| 元代后期 | 噶举派 | 势力上升，开始取代萨迦派地位 |
| 15 世纪初 | 格鲁派（黄教） | 新兴教派，宗喀巴弟子多次赴汉地朝见 |
| 明中后期 | 多派并立 | 各教派相对均衡，均主动与中央联系 |

> 📌 案例：三世达赖索南嘉措在西藏拉萨受排挤，跑到甘肃张掖，写信给明朝宰辅张居正（信中说"大国阁下张，名显如日月"），送藏传佛教礼物，希望借助明朝势力提高自身影响。

## 三、流动规模

### 3.1 元代

- 太原一路，**半年 185 次**往返，用马匹运输
- 元成宗大德九年至十一年，西藏到内地者 **850 余人**，用马 **1500 余匹**

### 3.2 明代

| 时期 | 一次朝贡人数 |
|------|:----------:|
| 成化年间 | 阐化王 462 人；后续达 1000—1470 人 |
| 弘治年间 | 数千人 |
| 宣德年间 | 较少 |
| 嘉靖年间 | 最多达 **4000 余人** |

### 3.3 违例朝贡问题

- 朝廷规定三年或两年一贡、每次限制人数
- 但实际上几乎全部"违例"——一年一贡、人数严重超限
- 原因：朝廷回礼按人数计算，人多赏赐多
- 另有**冒名朝贡**：临近藏区的汉族人从小习藏语，冒用藏族身份进京
- 明朝虽有政策限制，但执行不严格 → 本质是"怀柔远人"与地方利益之间的平衡

## 四、代表人物及其活动

### 4.1 凉州会谈关键人物

**萨迦班智达（萨班）**

- 1244 年受阔端之邀前往凉州
- 1247 年**凉州会谈**：代表吐蕃僧俗与阔端商讨西藏归顺蒙古事宜
- 萨迦派传承方式：**叔侄传承**（非父子传承）
- 携二侄前往：八思巴（10 岁）与恰那多吉（6 岁）
- 在凉州建**幻化寺**讲经说法，治愈阔端疾病，提升其藏传佛教信仰
- 1251 年圆寂于凉州
- ⚠️ 凉州会谈放在整个东亚历史格局中看，是**极其重要**的事件

### 4.2 元代帝师：八思巴（Phagpa, 1235—1280）

> ⚠️ 转录中"八思巴"被多次误识为"848""巴斯巴""阿斯巴"，以下统一为八思巴。

**生平概述**

| 年份 | 事件 |
|------|------|
| 1244 | 随萨班赴凉州，年 10 岁 |
| 1251 | 萨班圆寂，17 岁继任萨迦法主（宗教权力） |
| 1252—1253 | 在六盘山谒见忽必烈 |
| 1258 | 在开平府参加**佛道辩论**，战胜全真教道士 |
| 1260 | 忽必烈继大汗位，封八思巴为**国师** |
| 1264 | 忽必烈颁布《珍珠诏书》，赋予八思巴管理西藏僧俗权力 |
| 1269 | 创制**蒙古新字（八思巴文）** |
| 1270 | 晋升为**帝师**，封号**大宝法王** |
| 1280 | 圆寂于萨迦 |

**佛道辩论（1258 年）**

- 背景：全真教在华北扩张，将约 500 座佛教寺院改为道观，佛道争端激烈
- 辩论核心：《老子化胡经》真伪问题
  - 《老子化胡经》称释迦牟尼是老子化身 → 汉代以后为佛教本土化而作
  - 元代全真教利用此说将佛教纳入老子体系之下
- 八思巴以佛教代表身份参加，以博学和应变能力获胜 → 扩**大了藏传佛教在内地的影响**
- 藏传佛教有**辩经传统**（如色拉寺），源自印度，汉传佛教此传统已失

**八思巴文**

- 创制于 1264—1269 年，历时约八年
- 结合藏文字母和蒙古语语音特点
- 书写习惯遵循汉文和回鹘文规则
- 1270 年颁布全国 → 元朝票书、公文、印章、钱币、碑文均使用
- 元亡后废止，今仅见于藏族地区的印章和寺院门楣装饰

**其他贡献**

- 在五台山朝拜，开创藏传佛教与五台山的密切关系
- 主持佛事活动、翻译佛经、给蒙古王公子弟教学
- 居大都、五台山、临洮等地，大部分时间在内地活动

### 4.3 噶玛噶举派活佛转世制度的开创者

**噶玛巴西（Karma Pakshi，二世噶玛巴）**

- 出生于康区（今西藏江达县）
- 忽必烈与蒙哥同时召请 → 选择亲近蒙哥汗（疏远忽必烈）
- 蒙哥汗赐**金边黑色僧帽** → 噶玛噶举"黑帽系"地位崇高
- 蒙哥汗去世后，因与阿里不哥关系密切，被忽必烈囚禁多年
- 1264 年获释返回西藏
- 在西北（甘肃、宁夏、内蒙古）、漠北地区广泛传教
- 临终遗嘱直接促成了**藏传佛教活佛转世制度**的形成
- ⚠️ "五台山"辨析：噶玛巴西所到"五台山"可能是宁夏**贺兰山**（时称"北五台山"），非山西五台山

**攘琼多吉（三世噶玛巴）**

- 被认定为噶玛巴西的转世——活佛转世制度的正式实践
- 1332 年抵大都，经历元文宗 → 元宁宗 → 妥懽帖睦尔（元顺帝）三朝
- 在五台山朝拜禅修

**若必多吉（四世噶玛巴）**

- 出生康区洛隆县
- 在大都朝拜传法，经六盘山、北五台山、凉州等地建寺院

### 4.4 元代宫廷中的其他重要人物

**胆巴（Dampa，1230—1303）**

- 朵思麻（安多地区）人，八思巴同时代人
- 幼时赴西印度学法
- 至元七年与八思巴一同返回内地
- 为忽必烈传授藏传佛教**摩诃嘎拉（大黑天护法）**法门——主管军队的战神
- 1281 年主持第二次佛道辩论并获胜
- 因与宰相桑哥交恶被排挤回乡，桑哥失事后回大都
- 被贬潮州期间：为枢密使月历迷失之妻治病 → 月历迷失信奉藏传佛教 → 重修潮州**净德寺**，促**进藏传佛教在东南沿海传播**
- 元成宗铁穆耳即位后地位极高（1303 年），追封帝师

**达尼钦波桑波贝**

- 萨迦昆氏家族成员，因身份问题被流放**江南**（苏州、杭州、普陀山）
- 在江南 15 年，娶汉族女子生子
- 流放原因：远离元大都和西藏权力中心；杭州有萨迦派机构可约束
- 1298 年恰那多吉之子达玛巴拉去世后，萨迦昆氏家族后继无人 → 被召回萨迦继承法主
- 其后代多人担任元朝帝师

### 4.5 明代受封的藏传佛教领袖

**德银协巴（五世噶玛巴，1384—1415）**

- 明代获封**大宝法王**（元代此称号属八思巴/萨迦派，明改封噶举派）
- 永乐四年（1406 年）至南京觐见，在南京、北京、五台山大做法事
- 1409 年返回西藏

**昆泽思巴**

- 萨迦派，元代帝师贡嘎坚赞之孙
- 永乐皇帝因其道术高超召请，封**大乘法王**，地位仅次于大宝法王

**释迦益西**

- 格鲁派，宗喀巴弟子
- 永乐十二年（1414 年）入朝，封**大慈法王**
- 为明宣宗治病，建法缘寺
- 格鲁派由此与明朝建立直接联系

**侯显**

- 洮州（今甘肃临潭）藏族人，通晓藏汉双语
- 永乐年间代表朝廷出使乌斯藏、尼泊尔等地
- 据传与郑和同下西洋（有争议）
- 是明代藏人在中央机构任职的代表

**班丹扎释**

- 岷州大崇教寺僧人，永乐时担任大宝法王的译师
- 后奉朝廷旨意赴西藏**平息谣言**、稳定政局

## 五、元明藏地流迁的特点

| 特点 | 说明 |
|------|------|
| **规模大** | 元代驿站僧人众多；明代朝贡数千人，扰民现象严重但朝廷不严格限制 |
| **教派广泛** | 元代萨迦派主导 → 元末至明各教派均衡：噶举、萨迦、格鲁、甚至本教均有朝贡记录。唯宁玛派（红教）与政治关系最少 |
| **主动性高** | 与吐蕃时期（被动流落/内附）不同，元明藏人**主动来朝**——因朝廷给予高政治地位和经济利益（赏赐 + 私茶贸易） |
| **地域广** | 元代以北方为主（大都、五台山、西北）；江南有流放僧人。明代：南北两京、五台山、甘青川藏道、长江水路、大运河沿线皆有藏僧活动 |
| **社会影响深** | 涉及朝廷宗教政治事务、民间医疗、文学笔记小说（藏僧灵异故事增多） |

> 📌 关于**私茶贸易**：明代茶马贸易中官方压低马价抬高茶价，藏僧在回程途中大量采购内地廉价茶叶带回藏区获取暴利。为逃避监管，有人伪造批文、冒名朝贡。

## 六、历史影响

### 6.1 对藏族社会自身

- 13—17 世纪是西藏社会重大转折期
- 通过与中央王朝的联系，借助外部力量：**平息内部争端 + 推动内部改革**
- 元朝扶持萨迦派结束西藏近 400 年分裂
- 藏传佛教进入蒙古草原和内地，获得发展空间
- 源源不断的赏赐物资促进青藏高原财富积累和社会发展（如色拉寺为明朝赏赐资金修建）
- 14 世纪后藏文文献对内地历史叙事明显增多且详实 → 认知加深

### 6.2 对内地社会文化的影响

- **佛教艺术**：北京、五台山、杭州等地保留大量元明藏传佛教艺术遗迹（如杭州飞来峰——汉藏合璧风格）
- **工艺美术**：明代宫廷瓷器流行梵文、藏文纹饰；民间瓷器流行"八吉祥"等藏传佛教纹饰
- **文学**：元明笔记小说中藏僧形象和灵异故事增多
- **丰富了内地文化的多样性和生命力**

### 6.3 对统一多民族国家形成的影响

> "名时无外乎甚经济"（唐代诗句）→ 元明两代藏族与内地的深度交往，是在统一国家框架下重塑中华政治和文化理想的重要实践。

- 类似案例：元朝将南宋少帝赵显流放至萨迦出家；达尼钦波桑波贝被流放江南 → 双向人员流动反映出统一王朝的空间整合逻辑

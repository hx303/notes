# §6.3(2) 二阶常系数齐次线性微分方程

## 一、基本概念

> **二阶线性微分方程**：$y'' + P(x)y' + Q(x)y = f(x)$
>
> - $f(x) \equiv 0$ 时：**齐次**
> - $f(x) \not\equiv 0$ 时：**非齐次**
>
> 当 $P(x), Q(x)$ 为常数 $p, q$ 时，称为**常系数**方程：
> $$y'' + py' + qy = 0$$

## 二、特征方程法

> 设 $y = e^{rx}$ 为尝试解（因为指数函数的导数与其自身只差常数因子），
> 代入得 $r^2 e^{rx} + pr e^{rx} + q e^{rx} = 0$ → **特征方程**：
> $$r^2 + pr + q = 0$$

**结论**：$r$ 是特征根 $\iff$ $y = e^{rx}$ 是方程的解。

## 三、通解公式（$\Delta = p^2 - 4q$）

| 判别式 | 特征根 | 通解 |
|--------|--------|------|
| $\Delta > 0$ | 两个不等实根 $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| $\Delta = 0$ | 重根 $r_1 = r_2 = -\frac{p}{2}$ | $y = (C_1 + C_2x) e^{r_1 x}$ |
| $\Delta < 0$ | 共轭复根 $r = \alpha \pm i\beta$ | $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$ |

其中 $\alpha = -\frac{p}{2},\;\beta = \frac{\sqrt{4q-p^2}}{2}$。

### 重根情况推导

特征方程有重根 $r_1$ 时，一个解为 $y_1 = e^{r_1 x}$。
设另一解为 $y_2 = u(x)e^{r_1 x}$，代入原方程可得 $u'' = 0$ → $u = x$，
故 $y_2 = x e^{r_1 x}$。

## 四、解题三步法

1. **写特征方程**：$r^2 + pr + q = 0$
2. **求两个特征根**：$r_1, r_2$
3. **根据 $\Delta$ 选择通解公式**

## 五、典型例题

### 例1：$y'' + ay = 0$ 的通解

| $a$ | 通解 |
|-----|------|
| $a = 0$ | $y = C_1 + C_2x$ |
| $a > 0$ | $y = C_1\cos\sqrt{a}x + C_2\sin\sqrt{a}x$ |
| $a < 0$ | $y = C_1 e^{\sqrt{-a}x} + C_2 e^{-\sqrt{-a}x}$ |

### 例2：已知特解求方程

已知 $y = xe^{2x}$ 为特解 → 特征方程有重根 $r=2$
→ $(r-2)^2 = 0$ → $r^2 - 4r + 4 = 0$
→ 微分方程：$y'' - 4y' + 4y = 0$，通解：$y = (C_1 + C_2x)e^{2x}$

## 关键公式汇总

| 特征根类型 | 通解形式 | 记忆技巧 |
|-----------|---------|---------|
| 两不等实根 | $C_1e^{r_1x}+C_2e^{r_2x}$ | 两个指数叠加 |
| 重根 | $(C_1+C_2x)e^{rx}$ | 指数×一次多项式 |
| 共轭复根 | $e^{\alpha x}(C_1\cos\beta x+C_2\sin\beta x)$ | 指数×振荡 |

# §8.10 多元函数的极值与最值

---

## §8.10.1 无条件极值

### 一、定义

若 $z = f(x,y)$ 在 $(x_0,y_0)$ 的某邻域内有 $f(x,y) \leq f(x_0, y_0)$（或 $\geq$），则称 $f$ 在该点取得**极大值**（或**极小值**）。

> **例**：
> - $z = x^2 + y^2$ 在 $(0,0)$ 取极小值
> - $z = -x^2 - y^2$ 在 $(0,0)$ 取极大值
> - $z = xy$ 在 $(0,0)$ 无极值（马鞍点）

### 二、必要条件

> **定理1**：若 $z = f(x,y)$ 在 $(x_0, y_0)$ 存在偏导数且取极值，则
> $$f_x'(x_0, y_0) = 0,\quad f_y'(x_0, y_0) = 0$$

使偏导为 0 的点称为**驻点**。但驻点**不一定是**极值点（反例：$z = xy$ 的 $(0,0)$）。

### 三、充分条件（AC - B² 判别法）⭐

> **定理2**：若 $f$ 在 $(x_0, y_0)$ 邻域内有一阶和二阶连续偏导数，且 $f_x' = f_y' = 0$，令
> $$A = f_{xx}''(x_0, y_0),\quad B = f_{xy}''(x_0, y_0),\quad C = f_{yy}''(x_0, y_0)$$

| $\Delta = AC - B^2$ | 结论 |
|---|---|
| $\Delta > 0$ 且 $A > 0$ | **极小值** |
| $\Delta > 0$ 且 $A < 0$ | **极大值** |
| $\Delta < 0$ | **无极值** |
| $\Delta = 0$ | 无法判断，需另行讨论 |

> **理论依据**：利用了二次型的正定性。$f(x,y) - f(x_0,y_0) \approx A(\Delta x)^2 + 2B\Delta x\Delta y + C(\Delta y)^2$

### 四、求极值的标准步骤

1. 令 $f_x' = 0,\ f_y' = 0$，解出所有驻点
2. 求出 $A = f_{xx}''$，$B = f_{xy}''$，$C = f_{yy}''$
3. 在每个驻点处计算 $\Delta = AC - B^2$ 并判断

> **例1**（考研）：由 $x^2 + y^2 + z^2 - 2x + 2y - 4z - 10 = 0$ 确定 $z = f(x,y)$ 的极值
>
> 隐函数求偏导：$z_x' = \frac{1-x}{z-2}$，$z_y' = \frac{-1-y}{z-2}$
>
> 驻点 $(1,-1)$，对应的两个 $z$ 值分别 $A>0,A<0$ → 极小/极大

## §8.10.2 二元函数的最值

### 方法

函数在有界闭区域上连续 → 最值存在。最值可疑点：

1. **区域内部的驻点**（$f_x' = f_y' = 0$）
2. **边界上的可疑点**（转化为条件极值或代入边界）

计算所有可疑点的函数值，最大即最大值，最小即最小值。

> **常用技巧**：若实际问题中函数只有最大值且在区域内部取得，区域内只有一个驻点，则此驻点必为最大值点。

> **例2**：体积 $2\text{m}^3$ 的有盖长方体水箱，用料最省？
>
> 设长 $x$、宽 $y$，高 $h = \frac{2}{xy}$
>
> 面积 $S = 2(xy + \frac{2}{x} + \frac{2}{y})$
>
> 令 $S_x' = S_y' = 0$，得 $x = y = \sqrt[3]{2}$，此时用料最省（唯一驻点）

## §8.10.3 条件极值 — 拉格朗日乘数法

### 问题

求 $z = f(x, y)$ 在**约束条件** $\varphi(x, y) = 0$ 下的极值。

### 方法

> **拉格朗日乘数法步骤**：

**Step 1**：构造拉格朗日函数
$$L(x, y, \lambda) = f(x, y) + \lambda \cdot \varphi(x, y)$$

**Step 2**：令偏导为 0
$$\begin{cases} L_x = f_x + \lambda\varphi_x = 0 \\ L_y = f_y + \lambda\varphi_y = 0 \\ L_\lambda = \varphi(x, y) = 0 \end{cases}$$

**Step 3**：解方程组得可疑点，根据实际问题背景判断极值。

### 本质理解

从 $\varphi(x, y) = 0$ 可确定 $y = y(x)$，$z = f(x, y(x))$ 取极值：

$$\frac{dz}{dx} = f_x + f_y \cdot y' = 0,\quad y' = -\frac{\varphi_x}{\varphi_y}$$

$$\implies f_x\varphi_y - f_y\varphi_x = 0 \implies \frac{f_x}{\varphi_x} = \frac{f_y}{\varphi_y} = -\lambda$$

### 推广：多个约束条件

求 $u = f(x, y, z)$ 在 $\varphi(x, y, z) = 0,\ \psi(x, y, z) = 0$ 下的极值：

$$L(x, y, z, \lambda, \mu) = f + \lambda\varphi + \mu\psi$$

令五个偏导为 0，解方程组。

### 典型题型

> **例3**：求 $z = x^2 + y^2$ 与平面 $x + y - z = 2$ 之间的最短距离
>
> 目标：$d^2 = (x - u)^2 + (y - v)^2 + (z - w)^2$
>
> 约束：$z = x^2 + y^2$，$u + v - w = 2$
>
> 设切点后求距离

> **例4**：证 $x, y, z > 0$ 且 $x + y + z = a$ 时，$u = x^\alpha y^\beta z^\gamma$ 的最值
>
> $L = \alpha\ln x + \beta\ln y + \gamma\ln z + \lambda(x + y + z - a)$
>
> 得 $x = \frac{\alpha a}{\alpha+\beta+\gamma}$ 等

## 关键词汇总

| 概念 | 公式/判别 |
|------|-----------|
| 驻点 | $f_x' = 0,\ f_y' = 0$ |
| AC-B² > 0, A > 0 | **极小值** |
| AC-B² > 0, A < 0 | **极大值** |
| AC-B² < 0 | **无极值** |
| AC-B² = 0 | 不确定 |
| 拉格朗日函数 | $L = f + \lambda\varphi$（单约束）|
| 多约束 | $L = f + \lambda\varphi + \mu\psi$ |
| 最值 | 内部驻点 + 边界可疑点 → 比较函数值 |

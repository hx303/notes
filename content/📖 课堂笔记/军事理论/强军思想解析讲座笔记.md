# 强国必须强军——新时代强军思想解析

## 一、时代背景与核心问题

### 1.1 为什么要建设强大军队？

> 强国必须强军，军强才能国安。

- **伟大复兴的关键时期**：实现中华民族伟大复兴是近代以来最伟大的梦想，对军队来说就是**强军梦**
- **现实威胁**：仍面临**被侵略、被颠覆、被破坏**的风险
  - 校园内可能感受不到，但在相关部门工作的同学对此有深刻认识
- **底线手段**：强大的军队是**保底手段**，让敌人不敢轻易挑衅
- **国家责任**：建设强大军队是全国乃至整个国家的责任

### 1.2 时代特征

- 邓小平「和平与发展」的发展理念对当前仍有指导意义
- 2022年以来全球冲突频发（包括中美博弈）
- 新一轮**科技革命与军事革命**交互叠加
  - AI技术加速发展：一年甚至一个月就有新进展
  - 从手工书写到AI搜索研读——科技极大提高了效率
  - 在军事领域很多技术保密，普通人无法直接参与
- 2012年至今十年间经历诸多重大事件，**军队的强大保障了建设发展的稳定**
- 改革开放以来40年/30多年军队建设：职能概念、组织模式、发展阶段的认识都经历了变化

---

## 二、强军目标与战略安排

### 2.1 核心目标

> 建设一支**听党指挥、能打胜仗、作风优良**的人民军队
> ——2013年12月，习主席在解放军代表团全体会议上提出

### 2.2 三步走战略

| 时间节点 | 目标 |
|----------|------|
| **2027年** | 建军一百年（1927年南昌起义起算），基本实现机械化、信息化融合发展 |
| **2035年** | 基本实现国防和军队现代化（军事人员、组织形态、武器装备全面现代化） |
| **2050年** | 全面建成**世界一流军队** |

> 十九大报告明确提出：新时代的强军建设必须同国家现代化进程相一致。

### 2.3 总体国家安全观下的军队任务

- 维护**国家核心利益**、重大原则及主权安全
- 维护南海、台湾等方向的领土主权
- 维护海外利益安全
- 非战争军事行动：联合演习、维和、反恐救援等

### 2.4 实战化训练

> 台湾方向有动作、美国有动作 → 我方进行围岛演练
> 
> 2025年演习中，**台北101大楼被拍摄得非常清楚**，说明我们的准备非常充分

- 每次围岛演习让对手紧张
- 作战准备从经济到指挥体系全面改造

## 三、强军思想核心内涵（六个明确）

*根据2018年中央军委《关于加强新时代军队党的建设的决定》梳理*

### 3.1 明确党对军队的绝对领导

- **根本原则**：**党指挥枪**
- 党委统一的集体领导下的**首长分工负责制**
- **支部建在连上**
- 这是强军之魂，最核心的一条

### 3.2 明确军队要准备打仗

- 全军必须**聚焦备战打仗**
- 培养**「四有」新时代革命军人**：
  - 有灵魂
  - 有本事
  - 有血性
  - 有品德
- 全部心思向打仗聚焦，各项工作向打仗用劲

### 3.3 明确强军目标的三大要素

| 要素 | 定位 | 内涵 |
|------|------|------|
| 听党指挥 | **灵魂** | 绝对忠诚、绝对纯洁、绝对可靠 |
| 能打胜仗 | **核心** | 唯一体现价值的重要标准 |
| 作风优良 | **保证** | 传承红色基因、艰苦奋斗 |

### 3.4 明确政治优势

- 持续传承**红色基因**
- 全心全意为人民服务的**根本宗旨**
- 作风优良是区别于其他军队的显著标志

### 3.5 明确五位一体强军布局

| 维度 | 内容 |
|------|------|
| **政治建军** | 坚持党对军队的绝对领导 |
| **改革强军** | 组织形态、指挥体系现代化 |
| **科技强军** | 科技是**核心战斗力** |
| **人才强军** | 军官、士兵、文职人员三大队伍建设 |
| **依法治军** | 2021年十九届六中全会正式提出 |

### 3.6 明确斗争精神

> 维护国家核心利益、重大原则及主权安全、人民利益——各种风险都要**勇敢面对、坚决斗争，并取得胜利**。

## 四、军队改革与现代化建设

### 4.1 改革历程

| 时期 | 阶段特征 |
|------|----------|
| 1950年代 | 单一军种向**合成化**发展 |
| 1980年代 | 摩托化、机械化 |
| 1990年代后 | 机械化/半机械化创新 |
| 十八大以来 | **全方位改革**（思想、技术、作战体系） |

> 改革开放40多年来国家发展了300多项成就。对部队来说，改革是前进的**必要手段**。

### 4.2 指挥体制改革

```
军委管总 → 战区主战 → 军种主建

原18个集团军 → 整合为13个集团军
军-旅-营 三级体制（取消了军-师-团-营中的师团级，减少指挥层级）
```

**改革效果**：指挥更扁平、反应更快速。

### 4.3 新型作战力量

- 特种作战
- 立体攻防
- **两栖作战**
- **远海防卫**
- 战略威慑（战略预警、远程打击）

### 4.4 科技强军成果

- 航母建设（电磁弹射技术）
- 航空航天领域重大突破
- 六代战机研发
- 成都地区：航空航天展览（新会展中心）、丰富的人才和资源
- 核心产品可供参观

### 4.5 军队党的建设

- 2018年中央军委发布《关于加强新时代军队党的建设的决定》
- 全面加强军队党的政治建设、思想建设、组织建设、作风建设、纪律建设

## 五、人才与制度保障

### 5.1 军队人员三大构成

| 人员类型 | 说明 |
|------|------|
| **军官** | 指挥与管理骨干 |
| **士兵** | 战斗力的主体 |
| **文职人员** | 专业技术支撑（每年面向应届毕业生招聘，大三、大四时可关注） |

### 5.2 军事教育体系

- 开设军事理论课程，让青年人了解国防和军队
- **文职人员制度**：新的制度安排
- **军事教育方针**：培养有灵魂、有本事、有血性、有品德的新时代军人

### 5.3 依法治军

- **首次明确提出**：2021年11月十九届六中全会
- 核心转变：从「行政命令」转向「依法治理」
- 建设**中国特色军事法治体系**
- 坚持战斗力标准，按法制要求转变治军方式
- 以前可能靠行政命令，现在必须依法行事

>

> 历史充分证明，我们的人民军队不愧是**听党指挥的英雄军队**，忠心报国的英雄部队，为中华民族伟大复兴英勇奋斗的英雄军队。

## 七、核心要点总结

1. **强国必须强军**——国家安全与民族复兴的战略支撑，保底手段
2. **党对军队的绝对领导**——强军之魂（六个明确之首）
3. **备战打仗**——军队的根本职能和唯一价值标准
4. **五位一体**：政治建军、改革强军、科技强军、人才强军、依法治军
5. **三步走**：2027（百年建军）→ 2035（基本现代化）→ 2050（世界一流）
6. **斗争精神**：各种风险勇敢面对，坚决斗争，争取胜利

> 习近平：「人民军队必须强大，这是全国的重点任务。」

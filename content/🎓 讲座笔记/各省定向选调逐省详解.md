## 三、各省定向选调逐省详解

### 🥇 第一梯队（薪资最高）

#### 广东

| 项目 | 内容 |
| --- | --- |
| 报名时间 | 2025.12.1 |
| 笔试时间 | 2025.12.28 |
| 招录人数 | 1715 人 |
| 报名条件 | 党员/学生干部/校级奖励三者之一 |
| 笔试科目 | 行测 + 申论 |
| 笔试地点 | 北京、广州、深圳、佛山、东莞、湛江（多选） |
| 面试形式 | 线上结构化面试 |
| **薪资** | 省直22\~25万，深圳25万+，广州22万+，珠海/佛山23万+，东莞20\~27万，中山18万+ |

> 广东特色：行测有科学推理题（物理/化学/生物），申论有广东地方特色材料

#### 浙江

| 项目 | 内容 |
| --- | --- |
| 报名时间 | 2025.10.9 |
| 笔试时间 | 2025.11.8 |
| 招录人数 | 1018 人 |
| 报名条件 | 党员/学生干部/校级奖励/奖学金四者之一 |
| 笔试科目 | 《综合能力测试》（行测+申论合卷） |
| 笔试特点 | 行测50题60分钟60分 → 申论2题120分钟120分 |
| 面试形式 | 结构化面试 |
| **薪资** | 省直18\~20万，杭州20\~22万，宁波18\~20万，嘉兴/湖州15\~18万，金华15万，台州/丽水/舟山13\~15万 |

> 数量关系占比极高（15题/50题），相当于分数命脉；面试会考浙江相关常识（如"为什么红薯经济"）

#### 上海

| 项目 | 内容 |
| --- | --- |
| 报名时间 | 2025.9.8 |
| 笔试时间 | 2025.11.1 |
| 招录人数 | \~400 人 |
| 报名条件 | 党员+学生干部+校级荣誉+奖学金（需同时满足2项） |
| 笔试科目 | 行测 + 申论（行测含科学推理） |
| 面试形式 | **结构化小组** |
| 简历筛选 | 非常重要！通过率约10% |
| **薪资** | 约20\~22万 |
| 落户 | 有落户优势；上海本地高校/上海生源通过率更高 |

> 特色：先报名后筛简历，提交3个志愿，通过筛选才能笔试；面试结构化小组，分差极大

```
│ 国考      │ 2025.11.30   │ 行测 + 申论  │ 结构化/小组    │

```

### 编制类型

| 编制类型 | 经费来源 | 稳定性 | 说明 |
| ---- | ---- | --- | --- |
| **行政编制** | 全额财政 | ⭐⭐⭐⭐⭐ | 公务员身份，最稳定 |
| **参公事业编** | 全额/差额/自收自支 | ⭐⭐⭐⭐ | 待遇≈公务员，转行政编有机会 |
| **事业编制** | 视单位而定测试 | ⭐⭐⭐ | 需通过考试或提拔转行政编 |

> 差额拨款单位效益好时工资可能高于全额拨款（如5A景区、烟草d、银行、石油等垄断国企）

## 二、国考 & 中央选调 & 特招

### 国考

| 项目 | 内容 |
| --- | --- |
| 2026年招录 | 38,119 人（较上年略降） |
| 报名人数 | 约 300 万+ |
| 岗位分布 | 中央部委 \~1000人，其余为地方（税务占60%） |
| 学历要求 | 本科可报岗位少，硕士以上多 |
| 政治面貌 | 多要求中共党员 |
| 面试比例 | 1:5进面，1:2进入考察 |

### 中央选调

| 项目 | 内容 |
| --- | --- |
| 参与高校 | 34所（32所推荐200人，2所推荐50人） |
| 2026年招录 | 395 人 |
| 报名条件 | 应届 + 党员 + 校级奖励 + 学生干部 |
| 报录比 | 约 1:16（远低于国考某些岗位1:100+） |
| 流程 | 学校内部筛选 → 笔试 → 面试 → 考察 |
| ⚠️ 与国考冲突 | **央选和国考同一天笔试**（11月30日），只能二选一 |
| 面试差额 | 约1:2，常有调剂 |

### 中央选调职位表

* 每年12月中旬发布至学校
* 辅导员通知，就业办可查看
* 关注公众号「定向选调」
* 部委岗位：中纪委、中办、国办、外交部、发改委、国安部等

### 国考特殊岗位（GA/特招）

* 部分单位来校宣讲 + 面谈 → 笔试 → 面试
* 有涉外经历/海外居留经历者可能不通过

### 公务员职级小知识

| 学历 | 入职定级 | 晋升路径 |
| --- | ---- | ---- |
| 本科 | 一级科员 | 1年后→四级主任科员→三级→二级→一级（每级约2年） |
| 硕士 | 四级主任科员（副科级待遇） | 2年后→三级→二级→一级 |
| 博士 | 二级主任科员（正科级待遇） | 2年后→一级→四级调研员 |

> 实际晋升受职数限制，单位层级越高天花板越高；省直机关退休前可达一级调研员，县直机关副科可能就是天花板

### 国考岗位 vs 省考岗位

| 维度 | 国考岗位 | 省考岗位 |
| --- | ---- | ---- |
| 薪资 | 国家财政，发达地区可能低于地方 | 地方财政，发达地区更高 |
| 晋升 | 通道较窄（如税务系统） | 省内流动机会更多 |
| 天花板 | 单位层级高，天花板高 | 单位层级越低天花板越低 |

## 四、笔试备考策略

### 行测（副省级卷135题/120分钟）

| 模块 | 题量 | 目标正确率 |
| --- | --- | ----- |
| 言语理解 | 20 | 85-87% |
| 常识判断 | 15 | 75% |
| 数量关系 | 15 | 75% |
| 判断推理（图推/定义/类比/逻辑） | 35 | 75-80% |
| \<span style="color:#b3a2a2">资料分析\</span> | 20 | 85-87% |

### 行测刷题方法

```
4~5月：学完基础理论 + 各模块解题方法
6~7月：模块刷题，查漏补缺
8月：套卷刷题，限制时间
9月起：关注公告，各省特色题型突击
```

* **资料分析优先**：投入产出比最高，先攻克
* **套卷必计时**：到时间放笔，模拟真实考试
* **总结复盘**：分析哪个模块耗时多、错误多，针对性补强
* **刷题量目标**：行测50\~80套真题，申论50\~30套

### 申论评分标准（以40分为例）

| 档次 | 分数 | 特点 |
| --- | --- | --- |
| 天花板 | 29-30 | 总分论点精准，结构严谨，重点突出 |
| 上等 | 26-28 | 找准2-3个分论点，结构完整 |
| 中等 | 20-25 | 论点凑合，分论点不准确，结构松散 |
| 下等 | <20 | 论点不准，结构不全 |

> 申论分差一般不大（\~3.5分），但材料全部可摘抄引用

### 公基（公共基础知识）十大模块

1. 马哲原理
2. 毛泽东思想
3. 中特理论体系
4. 中国宪法与行政法
5. 公务员职业道德
6. 法律知识（宪法/刑法/民法/行政法）
7. 公文基础与写作
8. 经济与科技知识
9. 历史知识（近现代史为主）
10. 时事政治

### 各省特色题型须知

| 省份 | 特色 |
| --- | --- |
| 浙江 | 数量关系占比最高（15/50），决定成败 |
| 广东 | 科学推理（物理/化学/生物），10→5→10题波动 |
| 上海 | 科学推理 + 不定项选择 + 综合分析 |

| 天津 | 行测+公基+申论合卷 |
| 云南 | 纯选择题100题，含时政+法律+省情 |
| 内蒙古 | 主观题为主，无行测 |
| 湖北 | 行测可能考湖北相关常识 |

## 五、面试形式与备考

### 四种面试形式

| 形式 | 特点 | 采用省份 |
| --- | --- | ---- |
| **结构化面试** | 最常见，3题15分钟或2题10分钟 | 大多数省份 |
| **结构化小组** | 3人一组，互评互应，分差大 | 上海、部分税务 |
| **无领导小组** | 9人同场，分差极大 | 云南、部分岗位 |
| **面谈** | 半结构化，简历+提问 | 江苏名校优选、福建 |

### 面试备考节奏

* **笔试结束立即开始面试学习**，不要等出成绩
* 出成绩到面试约 **15\~20天**，这个时间完全够
* 表达能力弱的可以提前积累
* 表达能力强的不需要过早准备

### 面谈/考察注意

1. **学院环节**：考察同学关系、师生评价
2. **个人面谈**：自我介绍、简历提问、人岗匹配
3. **材料准备**：证书原件+复印件（务必提前准备）
4. 选调通常不公布面试成绩，直接通知是否进入下一轮

### 按竞争难度

* **最卷**：山东、河南、四川（成都）
* **较卷**：江浙沪、广东、北京、天津
* **中等**：湖北、湖南、安徽、福建
* **相对低**：东北三省、西北省份、西南边远省份

### 巡考策略

* 9\~12月：江浙沪+北京+广东+四川等，时间不冲突可巡考
* 12\~次年：各省联考只能选一个
* 全年约有 **9次** 考试机会
* ⚠️ 不要盲目巡考（如新疆、西藏），考上了不去浪费机会和金钱

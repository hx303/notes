导体、电容与电介质——辅导课笔记

🧑‍🏫 讨论形式：一对一辅导，涵盖导体静电平衡、电容器、串并联、电介质等内容。

一、导体与静电平衡（复习深化）

1.1 静电感应过程

将一个导体放入外电场 E0​ 中：

  1. 导体内的自由电子在电场力作用下移动
  2. 负电荷向与 E0​ 相反的方向聚集
  3. 正负电荷分离 → 导体内部产生感应电场 E′
  4. 感应电场与外电场方向相反
  5. 当 E内​=E0​+E′=0 时 → 静电平衡

💡 关键理解：静电平衡时导体内部场强为零，这是自由电子重新分布后的必然结果。

1.2 静电平衡的特征

特征说明
内部场强E=0
导体是等势体内部和表面各处电势相等
表面是等势面E 始终垂直于导体表面
电荷分布仅分布在导体表面（内部无净电荷）
表面场强E=ε0​σ​（紧贴表面的外侧）

1.3 导体表面场强的两种表达式辨析

这是最容易混淆的地方：

公式适用场景
E=ε0​σ​导体表面附近（一面有场，内部 E=0）
E=2ε0​σ​无限大均匀带电平面（两面都有场）

统一理解（叠加原理）：

将导体表面一小块 ΔS 视为无限大带电平面：

  • ΔS 自身产生的场：σ/(2ε0​)，两面各半
  • 除 ΔS 外的其他所有电荷产生的场：Eother​
  • 导体内：Eother​−2ε0​σ​=0 → Eother​=2ε0​σ​
  • 导体外：E总​=Eother​+2ε0​σ​=ε0​σ​ ✓

🔑 两者不矛盾——叠加原理统一了它们！

1.4 导体表面电荷分布与曲率

定性规律：曲率越大（越尖锐），电荷密度越大。

  • 尖端的曲率半径 R 极小 → 电荷密度 σ 极大
  • 尖端附近的电场极强 → 尖端放电（电晕放电）
  • 避雷针的原理正是利用尖端放电

📌 定量关系：对于孤立椭球导体，σ∝1/曲率半径

1.5 导体空腔

腔内有电荷：

  • 内表面感应出等量异号电荷 (−Q)
  • 外表面感应出等量同号电荷 (+Q)
  • 用高斯定理可严格证明

腔内无电荷：

  • 内表面无净电荷
  • 腔内 E=0（静电屏蔽）

1.6 接地问题

  • 接地 → 导体电势与大地电势相等（取 V=0）
  • 接地的实质：导体与大地交换电荷，直至等势
  • 常见问题：接地导体附近有带电体时，导体表面会产生感应电荷

1.7 静电平衡的建立时间

对于良导体（如铜），弛豫时间：

τ=σc​ε0​​≈1.5×10−19 s

极快！宏观上可视为「瞬间」完成。

二、电容与电容器

2.1 电容的定义

导体的电容：

C=VQ​

  • Q：导体所带电荷量（绝对值）
  • V：导体的电势（取无穷远为电势零点）
  • 单位：法拉（F），1 F=1 C/V

⚠️ 法拉是非常大的单位。实际器件中常用：

  • 微法：1 μF=10−6 F
  • 皮法：1 pF=10−12 F

电容的物理意义：反映导体容纳电荷的能力。电容越大，升高单位电势所需的电荷越多。

2.2 孤立导体的电容

孤立导体球（半径 R）：

V=4πε0​RQ​⇒C=VQ​=4πε0​R

例：地球半径 R≈6.4×106 m，C≈7×10−4 F=700 μF

2.3 电容器的定义

两个导体组成的系统，彼此绝缘：

  • 一个带 +Q，另一个带 −Q
  • 电容：C=UQ​，其中 U 为两导体间的电势差（电压）

三、常见电容器的电容公式

3.1 平行板电容器

（示意图：两块面积A、间距d的平行金属板）

C=dε0​A​

推导：

  1. 两极板间为匀强电场：E=ε0​σ​=ε0​AQ​
  2. 电势差：U=Ed=ε0​AQd​
  3. C=UQ​=dε0​A​

📌 讨论：A 越大、d 越小 → 电容越大。

3.2 圆柱形电容器

（同轴圆柱，内半径 R1​，外半径 R2​，长度 L，且 L≫R2​）

C=ln(R2​/R1​)2πε0​L​

推导思路：

  1. 用高斯定理求两柱面间电场：E(r)=2πε0​rλ​=2πε0​LrQ​
  2. 电势差：U=∫R1​R2​​Edr=2πε0​LQ​lnR1​R2​​
  3. C=UQ​=ln(R2​/R1​)2πε0​L​

3.3 球形电容器

（同心球壳，内半径 R1​，外半径 R2​）

C=R1​1​−R2​1​4πε0​​=R2​−R1​4πε0​R1​R2​​

极限情况：

  • 当 R2​→∞：C=4πε0​R1​ → 回归孤立导体球的电容 ✓

四、电容器的串联与并联

4.1 串联

![串联电容器]

特征：各电容器极板上电荷量相等（Q1​=Q2​=⋯=Q）

C总​1​=C1​1​+C2​1​+⋯+Cn​1​

电压分配： Ui​=Ci​Q​,U=U1​+U2​+⋯=Q(C1​1​+C2​1​+⋯)

📌 串联后总电容减小（小于任一个单独电容）。类似于电阻的并联。

4.2 并联

![并联电容器]

特征：各电容器两端电压相等（U1​=U2​=⋯=U）

C总​=C1​+C2​+⋯+Cn​

电荷分配： Qi​=Ci​U,Q=Q1​+Q2​+⋯=(C1​+C2​+⋯)U

📌 并联后总电容增大（等于各电容之和）。类似于电阻的串联。

4.3 串并联对比

串联并联
总电容公式C1​=∑Ci​1​C=∑Ci​
不变物理量电荷 Q 相同电压 U 相同
总电容大小小于最小者大于最大者
类比电阻并联串联

⚠️ 电容的串并联公式与电阻恰好相反！不要搞混。

五、电容器的能量

5.1 电容器储能

W=21​CU2=21​CQ2​=21​QU

推导（充电过程）：

  1. 充电过程中，极板间电压 u=q/C（q 为当前电荷量）
  2. 搬运 dq 电荷做的功：dW=udq=Cq​dq
  3. 从 0 充电到 Q：W=∫0Q​Cq​dq=2CQ2​=21​CU2

5.2 电场能量密度

平行板电容器：C=ε0​A/d，U=Ed

W=21​CU2=21​dε0​A​(Ed)2=21​ε0​E2⋅Ad=21​ε0​E2⋅V

→ 电场能量密度：

we​=VW​=21​ε0​E2

能量储存在电场中，而不是电荷本身。这是场的观点。

六、电介质简介

📝 本节为辅导课中提及的过渡内容，进入第六章”静电场中的导体和电介质”。

6.1 导体 vs 电介质

导体电介质（绝缘体）
自由电荷大量自由电子几乎没有
外电场中的行为静电感应（电荷重新分布，内部 E=0）极化（束缚电荷取向，内部 E=0）
电场能否穿透不能（静电平衡）能（但被削弱）

6.2 电介质对电容的影响

在电容器两极板间插入电介质：

C=εr​C0​

其中 εr​ 为相对介电常数（εr​>1），C0​ 为真空电容。

  • 电介质增大电容：C>C0​
  • 本质原因：电介质极化产生反向电场，削弱极板间电场 → 同样电荷下电势差减小 → C=Q/U 增大

6.3 极化机制（定性）

无极分子：正负电荷中心重合 → 在外电场中位移极化（感应电偶极矩）

有极分子：本身有电偶极矩（如 H₂O）→ 在外电场中取向极化

七、典型问题与解题技巧

7.1 导体相关判断

常见判断题：

  • ❌ “导体内部 E=0 意味着内部没有电荷” → 错，是无净电荷，但仍可能有等量正负电荷
  • ❌ “导体表面各处 E 相等” → 错，E=σ/ε0​，σ 未必均匀
  • ✅ “静电平衡时导体是等势体” → 对

7.2 电容计算通用步骤

  1. 假设两极板带电荷 ±Q
  2. 利用高斯定理（或叠加原理）求极板间的电场分布 E(r)
  3. 积分求电势差：U=∫E⋅dl
  4. 代入 C=Q/U，Q 约掉 → 得到仅与几何尺寸和介质有关的 C

7.3 串并联分析技巧

  • 先判断连接方式：并联 → 电压相等；串联 → 电荷相等
  • 串联分压公式（类似电阻并联分流）： U1​=C1​+C2​C2​​U总​
  • 并联分流公式：Qi​=Ci​U

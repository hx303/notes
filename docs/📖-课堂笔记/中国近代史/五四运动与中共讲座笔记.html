五四运动与中国共产党的建立

一、五四运动的背景与过程

1.1 1918年：振奋之年

事件内容
文人总统徐世昌当选总统——1912年袁世凯以来首次文人总统，此前全是武将总统
南北和谈1913年起南北战争不断，1918年开始和谈，民众期待稳定
一战胜利中国加入协约国，派出3000名华工 → 成为战胜国，国人激动不已

蔡元培校长宣布北大放假三天，师生穿统一服装在天安门前连续演讲三天。当时天安门没有扩音设备，但三天演讲效果极佳。

1.2 1919年：五四运动爆发

直接导火索：巴黎和会上山东半岛权益从德国转交日本 → 国人绝望

过程：

  1. 北大学生走上街头抗议
  2. 火烧赵家楼、痛打曹汝霖
  3. 学生被捕 → 陈独秀等教授愤怒，主动上街散发传单
  4. 北京警察逮捕了新文化运动领袖陈独秀

陈独秀狱中演讲：「知识分子有两个去处——要么监狱，要么书斋。在书斋里研究学问，在监狱里进行社会抗争。我们知识分子要随时准备走出书斋，走进监狱，然后又回到书斋。」

上海工人罢工（五四~6月3日）→ 工人阶级登上历史舞台——这是中共历史上解释五四运动的关键叙事

1.3 学生运动的分化

学运蓬勃发展 → 政府不敢管 → 学生发现警察拿枪（如《纪念刘和珍君》）
→ 学生开始分化：
  ├── 继续抗争
  ├── 投靠国民党
  └── 投靠共产党
→ 结论：政治力量非常重要，是改变社会的力量支柱

二、各方对五四的不同解读

2.1 胡适

  • 认为五四是中国「文艺复兴」的起点
  • 与欧洲文艺复兴类比：都强调以人为本

2.2 蒋介石

  • 批评五四过于激进
  • 认为五四带来了两种「西方主义」：
    • 自由主义——追求个人自由，导致国家如散沙，无法凝聚
    • 马克思主义——强调阶级斗争，地主与农民对立，社会分裂
  • 30年代推行新生活运动（蒋亲自参加，宋美龄也参加，搞了两年）
  • 美国历史学家易劳逸评价蒋的新生活运动报告：「缺乏实际工作」
  • 核心矛盾：批评了两种主义，但自己拿不出更好的方案来统一国家

2.3 毛泽东

  • 1939年高调纪念五四，写下《新民主主义论》——标志着毛泽东思想的成熟
  • 毛泽东是「能创造概念和词汇的人」
  • 两个核心能力：
    1. 能打仗（战争年代第一位）——遵义会议后土城战役虽失败，但随后证明了自己。1959年庐山会议仍念林彪之事
    2. 理论创新能力——新民主主义概念创造了一个全新框架

2.4 1937年王明归来与路线之争

  • 王明从莫斯科飞到延安，自称「昆仑山下来的神仙」
  • 提出口号：「一切服从于抗战，一切服从于统一战线」
  • 背后是苏联逻辑：希望国共合作牵制日本，避免苏联双线作战
  • Chiang’s slogan: 「一个军队，一个政党，一个主义」
  • Mao 的不同判断：抗战胜利后要大建国，中共需要统一自己的思想
  • 不自相矛盾：既要抗日，又不能公开违背苏联政策，也不能公开反对蒋

2.5 五四在中共党史上的特殊地位

五四运动在中共党史上的重要性在于：无产阶级登上了历史舞台。

所以”无产阶级领导的资产阶级革命”获得了合法性基础。

正因为如此，中共始终高调纪念五四——这关乎建党的合法性叙事。

就像长征（25000里，从江西到陕北），不容讨论、不容微词。每年的高调纪念具有”合法性技术”的性质。

三、中国共产党的建立

3.1 马克思主义在中国的传播之路

路径：

  1. 十月革命后，并非立即——而是1919年以后才真正开始大规模传播
  2. 李大钊通过日文报纸开始理解马克思主义
  3. 早期并非只有马克思主义一种思潮，无政府主义在当时也很时髦
    • 巴金（笔名来自巴枯宁 + 克鲁泡特金两位无政府主义者）早年信奉无政府主义，一生坚持”让世界充满爱”的理论
    • 文革后巴金反省自己，主张建立文革博物馆，强调说真话
  4. 李大钊从无政府主义转向共产主义：受俄国十月革命影响 → 工农的政府（多数人制度）是可以接受的

3.2 陈独秀的思想转变

早年信仰自由主义、民主国家理念
→ 反对民本主义，强调真正的民主
→ 五四前：热情歌颂美国总统威尔逊为"天底下最好的人"
→ 五四后：对威尔逊评价降低（"威大炮"，只会说大话）
→ 认识到：没有经济平等就无法实现真正民主
→ 转向马克思主义
→ 一生追求党内民主，试图实现"官兵一致"式的平等
→ 理想化难以实现 → 被淘汰 → 建托派组织 → 再次被淘汰

王国维：「可爱者不可信，可信者不可爱」 老子：「信言不美，美言不信」

3.3 早期党员构成

总数：约53-57人（有争议）

来源分析：

  • 留学生为主（留法、留日）
    • 留法勤工俭学：张志钊等人组织——拿路费、帮报考、介绍学校和工厂，可以一边工作一边读书。内地学生（湖南、四川为主）热情响应
    • 邓小平从成都附近来，父亲给了300大洋
    • 留日：名校毕业较多，如李汉俊（早稻田大学化工，曾任中共组织部长）
  • 地域分布：湖南、四川籍贯占比最大
  • 北大毕业生：大多为穷人

3.4 毛泽东的早期路径

毛泽东是湖南师范毕业，非北大高材生。

  • 好友蔡和森留法后写信邀请他，给了两个建议：
    1. 学点法语
    2. 筹备经费（湖南是内地省份，钱不多）
  • 毛泽东选择去北京，投奔杨昌济（中学老师），到北大当图书管理员
  • 中学老师袁老师教他古文（集大成版本）→ 脑子灵活
  • 关键优势：当李大钊、陈独秀给他讲马克思主义时，毛泽东马上用中国的”大同社会”等概念来对接 → 让李大钊印象深刻
  • 毛泽东并非不受同事欢迎，但他坚持与同学保持联系
  • 名言：「北京社会是一个大学校，有些人到国外去追求真理，我也要到社会的大学校里去寻求真理」
  • 1920年给蔡和森的信：德国式的革命无法实现，是”山穷水尽、诸路不通”的变形，并无更好的办法

3.5 共产国际的推动

人物/事件内容
威金斯基（维经斯基）共产国际特使，1919年后来华
来华目的打破帝国主义对苏联的封锁（苏维埃面临空前压力）
路径联络李大钊（曾写《庶民的胜利不是威总统之胜利》）→ 介绍陈独秀去上海
上海召集陈独秀邀请戴季陶（国民党宣传家）、张东荪（进步党宣传家）等人一同会见威金斯基
同期朝鲜共产党1920年成立、日本共产党1921年成立

3.6 各地共产主义小组

地点负责人特点
北京李大钊北大共产主义小组，人脉最广（刘仁静、张国焘、毛泽东受其影响）
上海陈独秀、李达陈独秀人脉广泛
长沙毛泽东在湖南建立
济南王志伟（陈独秀朋友的侄子）
武汉董必武（陈独秀老友）
广州威金斯基支持十月革命胜利后孙中山发电报祝贺，列宁回电”我们的胜利就是你们的胜利”
巴黎留学生组织

3.7 「问题与主义」之争

立场代表人物观点
多研究问题，少谈主义胡适实用主义思路（美国人不讲大意识形态，讲实际）
先谈主义，再解决问题李大钊主义是旗帜，先解决政治这个根本性问题

中国传统文化的影响：从关键观念出发再实施（朱熹”格物致知”、王阳明”致良知”），与儒家思想有心灵相通之处。胡适从美国回来，但中国传统文化根深蒂固——观点先行，先解决观念问题，再解决问题。

3.8 工人运动的艰辛

  • 北大共产主义小组：白天教工人识字，晚上讲组织
  • 上海：需加入青帮才能发动工人（青帮有200人规模），但1927年大部分失败后清理门户很困难

3.9 中共一大（1921年7月）

  • 1921年7月23日，上海开会
  • 代表：13人，代表约57个党员
  • 日期争议：最终确定为7月23日（当年有学者花几十年研究确定）
  • 周佛海带夫人参加，在房子里开会（音乐表演在大厅）
  • 重要代表后来的去向分化很大：
    • 毛泽东、董必武 → 登上天安门
    • 刘仁静 → 脱党（1949年后仍为名流，在湖南大学、武汉大学工作）
    • 张国焘 → 脱党→ 国民党 → 成为中共在台湾的对手
    • 李汉俊等 → 较早牺牲
    • 还有一些在90年代才被重新认识

四、马克思主义基本理论

4.1 马克思对现代社会的批判

核心问题：现代社会是为资本而生产，不是为需求而生产

  • 传统社会：生产多少就消费多少
  • 现代社会：生产为了创造收入，而非满足需求
  • 人的异化：我们被培养成「工具」而非「人」，按照劳动力需求来培养
  • 市场经济规律：是人为设定还是自然形成？失业问题带来压迫感
  • 马克思的判断：无产阶级会日益觉醒

4.2 马克思主义的分化

马克思之后分出两种路径：

派别路径代表
社会民主主义议会斗争，帮工人维权德国社会民主党
列宁主义职业革命家队伍，暴力革命苏联共产党

德国社民党的问题：工人阶级日子不错 → 没人支持革命 → 改为议会斗争维权 → 被列宁视为问题所在

4.3 鲍德里亚的符号价值理论

马克思 → 商品有两种价值：使用价值、交换价值

鲍德里亚 → 现代消费社会增加第三种：符号价值

  • LV包 ≠ 普通包的社会意义
  • 消费社会中，评价标准不是「你是谁」，而是「你拥有什么符号」
  • 幸福被定义为外在的待遇和物质拥有

4.4 列宁主义

  • 革命需要职业革命家队伍（共产党）
  • 需要能够获得人民支持的意识形态
  • 共产党双重属性：先进性 + 代表人民利益

4.5 世界体系论（沃勒斯坦）

  • 发达国家的发达 ≠ 自我努力
  • 发达国家的发展伴随着对贫穷国家的剥夺
  • 发达与贫穷是相互依存的

4.6 西方马克思主义（葛兰西）

  • 意大利革命失败 → 被关押20年 → 监狱中写《狱中札记》
  • 核心问题：为什么俄国成功而意大利失败？
  • 发现：意大利深受资产阶级思想影响，知识分子满脑子资产阶级思想 → 革命失去合法性
  • 提出文化霸权理论

4.7 「不忘初心，牢记使命」的历史逻辑

欧立德（哈佛历史学家）：一个政体最多350年。 第一代人开创天下极其辛苦（如朱元璋），越往后越难保持。 我党为何反复开思政课——不想好不容易打下的政权被下一代人”卖掉”。

五、国共合作（1923-1927）

5.1 合作的背景

方需求
苏联共产党人数太少（几十人），需要找实力派合作
国民党孙中山需要苏联支持建军
共产党需要合法活动空间

曲折过程：

  1. 苏联先找吴佩孚（直系实力派）
  2. 后转向孙中山（革命老搭档，但非实力派）
  3. 马林提出「平行合作」（国共各自为政）被拒——孙中山认为共产党是”一群没有政治经验的学生”
  4. 马林转而提出「党内合作」：共产党员以个人身份加入国民党

5.2 党内合作的博弈

孙中山的态度：

  • 自信三民主义包含了共产主义 → 共产党可以作为国民党的一个派系
  • 希望共产党给国民党做纪律榜样

马林召开西湖会议：邀请李大钊等商讨 → 陈独秀、李大钊最初反对（这类似于一个”军团”模式）

1923年中共三大：确立国共合作 → 1924年1月国民党一大正式确立

反对声音（国民党元老：张继、谢持等）：

「前五人后不来者」——既是共产党又是国民党，要尊重国民党党籍又要忠诚共产党，无法平衡

李大钊的回应：

「我们加入国民党是为了革命这个目标。」 「兄弟深恐在这个时候埋下焦虑和担子。」

5.3 合作成果

  • 1924年1月：国民党一大，黄埔陆军军官学校同时建立（口号：「到黄埔去」）
  • 苏联提供一万余名士兵的装备和大量机关枪
  • 共产党获得合法活动空间：到1927年大革命失败时，党员达6万人，成为第二大政党
  • 在各地建立工会、农会，迅速发展壮大

5.4 埋下的隐患

「种子从一开始就埋下了焦虑和烦恼」——李大钊在国民党一大上的预感，后来被历史验证。

六、中共早期纲领

6.1 最高纲领与最低纲领（1922年中共二大）

纲领内容
最高纲领实现共产主义（永恒目标）
最低纲领反帝反封建，争取民族独立和人民解放
  • 二大分析了中国是半殖民地半封建社会的性质
  • 写了关于殖民地民族国家的文章，明确中国在世界无产阶级革命运动中的地位

江泽民十五大：「中国共产党始终是最高纲领和最低纲领的统一论者」

6.2 建党的历史意义（四个「新」）

  1. 坚强的政党——革命有了领导核心
  2. 科学的革命理论——马克思主义指导
  3. 新的革命方法——自下而上的群众运动（以前都是自上而下）
  4. 世界无产阶级革命的一部分——与苏联及其他社会主义国家联合

七、延伸思考

7.1 毛泽东的「新民主主义」理论

  • 半殖民地半封建社会 → 资产阶级民主革命 → 社会主义
  • 资产阶级无法完成革命（妥协性） → 共产党领导完成
  • 理论内在矛盾：新民主主义应是长期过程，但1949年后仅用三年左右就完成了社会主义改造
  • 刘少奇一直坚持新民主主义是漫长过程的观点

7.2 革命的可能性问题

  • C曲线理论：普遍不满 ≠ 必然革命
  • 革命需要两个条件：
    1. 先锋队（共产党在前面开路）
    2. 照顾人民利益（人民才会跟随）
  • 食堂不好吃 ≠ 造反；川菜好吃 ≠ 归附

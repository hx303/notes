鸦片战争与中国近代史的开端

📅 2026年3月23日 ⏱ 约3小时课程

一、引言：从波德莱尔谈起

课程从法国诗人波德莱尔（Charles Baudelaire）切入。他是现代主义文学的先驱，其作品标志着现代性概念 的诞生。波德莱尔曾写过酒、鸦片、大麻三样东西，希望通过这些感官体验来理解现代性。

“一切坚固的东西都烟消云散了。” —— 马克思

马克思这句话揭示了近代以来的最大特点：传统价值的瓦解。波德莱尔试图通过鸦片的感受来捕捉这种现代性体验。而鸦片在中国的大规模传播，正是中国近代史的重要起点。

二、全球背景：白银流动与世界贸易

2.1 地理大发现与白银革命

  • 1492年 ：哥伦布发现美洲，西班牙获得大量白银
  • 在玻利维亚波托西发现银山，白银源源不断流入欧洲
  • 西班牙国王查理一世的年收入是英国伊丽莎白女王的8倍

白银大量流入导致欧洲价格革命 （通货膨胀），英法等国采取重商主义 政策，鼓励出口、限制进口，发展制造业，将西班牙的白银赚到自己手中。

2.2 中国的货币需求

明清时期，中国使用白银作为主要货币：

  • 铜钱用于日常交易，但交税必须用白银
  • 白银便于携带，适合大宗贸易和长途运输
  • 明朝”一条鞭法”后，基本上用白银交税

火耗问题： 百姓交税时，散碎银两需熔铸成银锭，这个过程会有损耗，称为”火耗”。官员常借机多收，成为贪腐来源。雍正曾试图禁止，但效果有限。

2.3 白银流动的转向

18-19世纪，大量白银从欧洲流入中国，换取丝绸、瓷器、茶叶。但19世纪初拉美独立运动 后，白银供应链断裂。英国需要找到一种能与白银对冲的商品——鸦片 。

三、鸦片贸易与白银危机

3.1 鸦片贸易的兴起

1800年起 鸦片贸易逐渐增多 1839年 鸦片贸易达到45,000箱

3.2 白银外流的社会影响

白银大量外流导致银贵钱贱 ：

  • 原本1两白银 = 1000文铜钱
  • 白银减少后，1两白银可能需要1500-1800文铜钱
  • 偏远地区（云南、广西、四川）兑换率更高

百姓日常使用铜钱，但交税必须用白银。银价上涨意味着百姓需要拿出更多铜钱来换取同样数量的白银交税，尤其是偏远地区农民压力更大。社会矛盾加剧，成为农民运动的重要诱因。

四、闭关政策与朝贡体系

4.1 中国是否”闭关自守”？

近代以来中国并非完全闭关，而是存在以朝贡体系 为基础的贸易往来：

  • 周边国家（日本、朝鲜、越南等）定期向中国皇帝朝贡称臣
  • 中国皇帝回赐更多礼物
  • 朝贡使团同时进行商品贸易

怀柔远人： 这种贸易建立在”天朝上国”的政治框架下，而非现代意义上的平等自由贸易。

4.2 马戛尔尼使团（1793年）

英国派马戛尔尼来华，希望与中国进行自由贸易。但双方在跪拜礼仪 问题上无法达成一致，最终不欢而散。

从欧洲视角看，各国大小相当，形成平等贸易关系；从东亚视角看，中国太大，其他国家太小，无法平等。这是两种不同世界观的碰撞。

五、禁烟决策与虎门销烟

5.1 两种主张

面对白银外流，清廷内部有两种意见：

  1. 弛禁派（黄爵滋） ：主张在国内种植鸦片，自产自销，避免白银外流
  2. 严禁派（林则徐） ：主张严禁鸦片，打击吸食和贩卖

“若犹泄泄视之，是使数十年后，中原几无可以御敌之兵，且无可以充饷之银。” —— 林则徐奏折

5.2 虎门销烟（1839年）

林则徐被派往广东禁烟：

  • 要求鸦片商人限期交出鸦片
  • 通过十三行（清政府特许的贸易机构）施压
  • 在虎门海滩挖坑，用石灰和开水销毁鸦片

六、鸦片战争进程（1840-1842）

第一阶段 1840年6月 - 1841年1月：英军抵达天津，道光将林则徐革职 第二阶段 1841年1月 - 1841年5月：虎门战役，关天培战死，广州沦陷 第三阶段 1841年8月 - 1842年8月：英军沿长江北上，镇江战役，南京条约签订

6.1 军事差距

清军与英军存在巨大差距：

  • 武器 ：清军使用大刀、长矛、鸟枪（射程约100米）；英军使用先进火炮和燧发枪
  • 战术 ：清军长期不打仗，训练废弛
  • 第二次鸦片战争时差距更大 ：英法联军使用阿姆斯特朗炮，八里桥之战几乎”屠杀”了蒙古骑兵

6.2 镇江战役

镇江战役是最惨烈的一场战斗。驻防镇江的满洲八旗男女老少齐上阵，几乎全部战死。他们”分房驻防”，祖坟、财产都在当地，一旦失败就全完了，因此拼死抵抗。

八旗制度 ：清朝以少数满人统治庞大汉人群体，将满人分散驻扎在重要城市。满人享有特权（从内务府领工资），但入关后很快汉化，到清末已与汉人无明显差异。

七、南京条约（1842年）

7.1 主要内容

  1. 赔款 ：2100万银元（烟价600万 + 商欠300万 + 军费1200万）
  2. 割地 ：香港岛割让给英国
  3. 五口通商 ：开放广州、厦门、福州、宁波、上海
  4. 协定关税 ：关税由双方协商确定（约5%），中国丧失关税自主权
  5. 取消十三行 ：外国人可以直接与中国商人贸易

7.2 当时人的看法

当时的中国人并没有”不平等条约”的概念。道光皇帝郁闷的是割地赔款”无颜面对列祖列宗”，而非民族主权被侵犯。官员们觉得与英国女王签订协约有失体面，但没有被侵略、被剥夺的感觉。

“不平等条约”的概念是民国以后 才逐渐形成的。1943年抗战期间，中国与同盟国签订新约，废除不平等条约。

八、半殖民地半封建社会

8.1 概念辨析

“半殖民地半封建社会”是中共1922年提出的概念。更准确地说：

  • 半殖民地 的另一半是半独立 （中国仍有自己的政府）
  • 半封建 的另一半是半资本主义 （西方入侵促进了中国资本主义发展）

史学界曾建议改为”半殖民地半资本主义社会”，但”半殖民地半封建社会”的表述已约定俗成。

8.2 社会特征

政治层面：

  • 西方帝国主义是中国近代的主导力量
  • 近代所有成功的政治力量背后都有西方支持

经济层面：

  • 传统自给自足经济被破坏
  • 但封建地主阶级仍然存在
  • 民族资本主义有所发展但不充分
  • 沿海地区受西方影响大，内陆影响小

社会层面：

  • 政治经济发展不平衡，四分五裂
  • 农民日益贫困化

九、中国近代化的困境

9.1 国有经济的问题

以张之洞创办汉阳铁厂 为例：

  • 张之洞从德国进口高炉，但未考虑铁矿石来源
  • 后来发现大冶铁矿，但矿石品质不适合
  • 只能从澳大利亚进口矿石，年年亏损
  • 最终交给盛宣怀经营，改为汉冶萍公司

国有经济往往存在政绩观问题——官员希望快速出成绩、升官，而非长远经营。盛宣怀接手后，将国有企业变成自己的企业，成为首富，也引发”有才干的人没道德”的争议。

9.2 四分五裂的局面

1930年代的中国实际上处于军阀割据状态：

  • 蒋介石只控制上海和江浙地区
  • 张学良控制东北
  • 阎锡山控制山西
  • 冯玉祥控制西北
  • 四川更是军阀林立，实行”防区制”，预征税收到几十年后

中国需要统一，但统一不能学美国（联邦制），只能学苏联——一个政党、一个军队、一个主义 。

9.3 国民党的失败

蒋介石曾发起新生活运动 ，希望培养国民的国家意识。但两年后他自己总结：运动流于形式，缺乏实际内容。

历史学家易劳逸评价：国民党失败的原因就是太容易做表面工作，缺乏实际内容和执行力。

9.4 农民的困境

西方学者形容中国农民”像水摸了脖子”——任何一个小浪花都可以让他们倒下。近代从小政府变大政府，需要征税办教育、修路等，所有负担都落在农民身上。

冯友兰曾写信给蒋介石，希望推动民主改革。蒋读后感动流泪，但毛的评价是：“民主无量，独裁无胆” ——喜欢民主的表象，但没有容人的雅量。

📋 本讲核心要点

  1. 鸦片战争的根源 ：全球白银流动、中英贸易逆差、拉美独立运动切断白银供应
  2. 白银外流的后果 ：银贵钱贱，农民负担加重，社会矛盾激化
  3. 朝贡体系vs自由贸易 ：两种世界观的碰撞，马戛尔尼使团的失败
  4. 南京条约的意义 ：中国近代史开端，但当时人并无”不平等条约”概念
  5. 半殖民地半封建社会 ：半独立+半资本主义，沿海与内陆发展极不平衡
  6. 近代化困境 ：国有经济效率低下、军阀割据四分五裂、农民贫困化
